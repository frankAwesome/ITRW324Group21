Details

High Quality Video Calling!

Video call any connected user at high speed rates without using any isp provided bandwidth (excl. VPN users connected via the internet)

- Users connected locally can expect minimal latency and smooth streaming

- A webcam & microphone is required

Video Call Connections

Connect to any user by entering the respective IP number in the text-field. The port should be '100' by default. Advanced settings can be found, and adjusted on-screen at the user's preference. Alternatively a user can start a connection by entering their own IP address in the text field and clicking on the 'start button'. A connection can be terminated at any time by clicking the 'Disconnect' button on-screen.
