Details

Convenient Voice Messaging

Record and listen to voice messages without using any isp provided bandwidth (excl. VPN users connected via the internet)

- Advanced customization allows users to fine-tune their desired record settings

- A microphone is required

Recording

Record a voice-note of any length by selecting the desired input device from the listbox (connected microphones with installed, up to date, drivers should be listed) The default settings can be used for a standard setup. A user may fine-tune these depending on preference.
