File Transfer

Details

Instantly Transfer Files To Any Connected User!

Transfer any file to any connected user at high speed rates without using any isp provided bandwidth (excl. VPN users connected via the internet)

- Users connected via ethernet cables can expect a rate of up to 100mbps

- Wirelessly connected users can expect a rate of up to 5.5mbps

Transfers

Follow the on-screen instructions in 'File Transfer' tab to transfer/allow any file to/from another connected user. Any user can host a connection to enable transfers, allowing any other connected used to upload/download files to/from the host. All transfers can be found, by default, in the 'Transfers' folder in the application directory. Alternatively, a user/host can create/choose a preffered directory
